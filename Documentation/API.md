# 天気予報を取得し、表示する
`YumemiWeather`のAPIを使用して、天気予報を取得しましょう。  

Simple ver  
`public static func fetchWeatherCondition() -> String`  
[APIの概要](YumemiWeather.md)

## 課題
- ボタンをタップして天気予報を取得する
- 天気予報を画面に表示する
